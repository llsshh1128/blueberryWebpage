Inorder Successor in BST (285)Â¶

Recursion

Given a binary search tree and a node in it, find the in-order successor of that node in the BST.

Note: If the given node has no in-order successor in the tree, return null.


Solution

  1. Three cases: has right subtree, no right subtree but is others’ left subtree, no right subtree and is not others’ left subtree
  2. Recursion to change root and find the suitable root

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */

public class Solution { 
	public TreeNode successorBST(TreeNode root, TreeNode p) {
        if (root == null)
            return null;

        if (root.val <= p.val) {
            return successorBST(root.right, p);
        } else {
            TreeNode left = successorBST(root.left, p);
            return (left != null) ? left : root;
        }
	}
}

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */

public class Solution { 
	public TreeNode predecessorBST(TreeNode root, TreeNode p) {
        if (root == null)
            return null;

        if (root.val >= p.val) {
            return predecessorBST(root.left, p);
        } else {
            TreeNode right = predecessorBST(root.right, p);
            return (right != null) ? right : root;
        }
	}
}
